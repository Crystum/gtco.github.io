How to join?

Open Space Bar.

Find "Notepad".

Press on the Notepad with right button on Your mouse and open Notepad "As Administrator".

Now press on "Open" and find a file named "Hosts".

Add into this file the IPs of our server:

51.79.145.101 growtopia1.com

51.79.145.101 growtopia2.com

Save the file by combo of Ctrl+S or just press on "Save".

Close the file and open Growtopia."

Enjoy!"

Open "App Store" and download "Surge 3/4"

As you downloaded, open the "Surge 3/4" and scroll down until You see "DNS". As You have seen the "DNS", tap on the "Local Mapping"

Press on "Add new", Now You should make 2 Files:

In the "Domain" You should write "growtopia1.com" and put into "Value" the IP of the server.

Now, as You added the first File, You should make the second, last one:

Again press on "Add new". In the "Domain" You should write "growtopia2.com" and put into "Value" the IP of the server.

Now, as You made 2 Files, press on "Done" button

Do not be afraid, You should allow to Surge 3/4 add a VPN. Press on "Allow"

To find the "Hosts" File, open Finder, and in Finder’s menu bar, select Go > Go to Folder.

In the box, type the following location /private/etc/hosts and press "Return" button.

A new "Finder" window will open and your Mac’s hosts file will be selected. Click and drag it out of the "Finder" window and drop it on your desktop. This will let us freely edit the file.

To open it, simply double-click and it will display the file's contents in TextEdit. Add the following lines:

51.79.145.101 growtopia1.com

51.79.145.101 growtopia2.com

Save the file and put it back where you found it.

Run Growtopia.

Enjoy!

How to join on Android?

Open Your Web-browser and download "Virtual Hosts", by using this link - Tap there to download

As You downloaded the "Virtual Hosts", You should download the hosts file - Tap there to download

Now, as everything is downloaded, open Virtual Hosts and click on "Re-select hosts file".

Find on file with hosts, which You downloaded a while ago. Press on it.

Now, as You added the host's file, Your VPN is turned on

Run Growtopia

Enjoy!
